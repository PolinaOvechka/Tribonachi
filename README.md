# Tribonacci

Windows Forms приложение на C# для работы с числовыми последовательностями:
- Генерация последовательности Трибоначчи с расчётом отношений соседних членов
- Сравнение последовательностей Фибоначчи и Трибоначчи
- Визуализация графиков через класс Graphics

## Как запустить

### Через Visual Studio
1. Откройте файл `Tribonachi.slnx`
2. Нажмите `Ctrl + F5` (`Ctrl + Fn + F5`) для запуска без отладки
